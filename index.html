Переезд с ChatGPT в Claude

Всё, что ChatGPT знал о вас — переносим в Claude. Контекст — это король.

1
Открыть настройки ChatGPT

Зайдите в Settings → Personalization → Memory и скопируйте промт для экспорта памяти.

2
Вставить промт в ChatGPT

Вставьте в любой чат ChatGPT. Он соберёт всё, что знает о вас, в структурированный текст.

3
Открыть раздел импорта в Claude

Прямая ссылка на настройки: claude.ai/settings/capabilities → Start Import → вставьте память → Сохранить.

4
Создать Проект «О себе»

Создайте в Claude проект с вашей базой: ДНК Бизнеса, ДНК Клиента, ToV, продукты, кейсы.

💡

Прямая ссылка на импорт: claude.ai/settings/capabilities — сохраните в закладки, пригодится при настройке.

Метапромтинг

Когда лень каждый раз придумывать промт — даёте максимум информации и просите Claude самого сформировать задачу.

// Шаблон метапромта Вот вся информация обо мне / проекте / задаче: [вставьте всё, что знаете] Что мне нужно сделать: [опишите задачу своими словами] Задай мне 5-7 уточняющих вопросов, которые нужны тебе для решения этой задачи. Когда получишь ответы — объясни, что ты понял, и скажи, что ещё нужно. Потом идём работать.
💡

Суперсила: Можно дать Claude цепочку: «Сначала сделай исследование, потом задай вопросы, потом напиши метапромт» — всё в одном сообщении, без вашего участия.

⚡

Живая демонстрация

📌 Пример: «Тренажёр-задрочка» для Артёма Яешника

Задача: создать тренажёр, который поможет выработать иммунитет к обидным шуткам о фамилии — через психотерапию + юмор.

  • Роль сотрудника: пересечение психотерапевта + комика
  • Deep Research: как фамилия влияет на психику и самооценку
  • Уточняющие вопросы: реакция на подколы, уровень самоиронии, аудитория
  • Формат: список задрочек, жёсткий и мягкий стёп, финальный рефрейм
  • Бонус: текст песни в стиле Pink Floyd + стили для генерации в Suno
🎓

Вывод из демо: Разница между «напиши обзывалки» и детальным метапромтом — это разница между шаблонным мусором и персонализированной программой. Всё решает контекст.

Модели Claude

Claude — компания с тремя сотрудниками разного уровня: Junior, Middle и Senior. Выбирайте под задачу.

МодельУровеньЗадачиСкорость
Haiku
БыстрыйПростые задачи, автоответчики, короткие тексты⚡⚡⚡ Очень быстро
Sonnet
Рабочая лошадка80% повседневных задач: тексты, анализ, лендинги, контент⚡⚡ Быстро
Opus
ТопСтратегия, методологии, сложный анализ, планирование⚡ Медленно, думает долго

🎛️ Режимы усилий

  • Adaptive: модель сама решает, сколько думать
  • High: рекомендуемый рабочий режим
  • Max: 5 сценариев на каждый ответ, ест токены в 5x

💰 Стратегия

Opus создаёт план → Haiku пишет пост. Экономия токенов в 10x.

Задачи на деньги — самая умная модель. Остальное — Sonnet/Haiku.

Контекстное окно

Контекст — король. Нейронки работают как СДВГэшные подростки — долго общаешься, забывают начало.

📊 Размеры

  • Стандартный чат: базовый лимит
  • Claude Code (Opus 1M): ~750 страниц текста
  • Файлы: хранятся локально на вашем компьютере

📁 Типы памяти

  • Memory (настройки): о вас между чатами
  • Проекты: постоянный контекст по проекту
  • Локальные файлы: можно передать другой нейронке
💻

Почему Desktop, а не Web: Даёте Claude доступ к папке → он хранит память локально → 1M токенов в сессии. Плюс: ни одной утечки данных за всё время существования.

Claude Code

Чёрный экран с белыми циферками. Не страшно — это самый мощный способ работать с Claude.

🚀 Что умеет

  • Создавать файлы и папки на компьютере
  • Делать сайты, скрипты, мини-приложения
  • Принимать цепочку задач в одном сообщении
  • Работать автономно, пока вы пьёте кофе

⚙️ Первая настройка

  • Создайте папку claudecode на компьютере
  • Дайте Claude доступ к ней
  • Включите Bypass Permission Mode если нужна автономность
🔐

Bypass Permission Mode: Settings → Claude → ползунок. Теперь Claude не будет каждый раз спрашивать разрешения — удобно для долгих задач.

Живая практика

Пока мы на уроке — делаем сразу. Три задачи прямо сейчас.

1
Установить MacWhisper и настроить горячую клавишу
⏱ 10 мин

Скачайте, установите, выберите языковую модель (русский), скачайте её локально. Назначьте горячую клавишу для голосового ввода.

2
Перенести память из ChatGPT в Claude
⏱ 5 мин

Используйте промт из урока. Скопируйте память → импортируйте через claude.ai/settings/capabilities → создайте Проект «О себе».

3
Написать первый промт по 5-элементной формуле
⏱ 10 мин

Возьмите реальную задачу из бизнеса. Напишите промт: Роль → Контекст → Задача → Ограничения → Формат. Потом повторите через метапромтинг и сравните.
