База база

💬 Что такое промт и из чего он состоит

Промт — это не «вопрос нейронке». Промт — это техническое задание сотруднику, у которого нулевой контекст о тебе. Если задание говно — результат говно. Без вариантов.

РОЛЬ → КОНТЕКСТ → ЗАДАЧА → ОГРАНИЧЕНИЯ → ФОРМАТ

❌ Как 90% людей пишут

напиши пост про мой курс

Нейронка выдаёт усреднённую херню, потому что у неё нет ни роли, ни контекста, ни цели.

✅ Как надо

Роль: ты маркетолог-копирайтер, спец по прогреву в соцсетях.
Контекст: у меня курс по AI-маркетингу, ЦА — предприниматели 28-45, боль — отстают от конкурентов.
Задача: напиши пост на 1500 знаков для Threads.
Ограничения: без канцелярита, живой язык, мат точечно.
Формат: хук → история → инсайт → CTA.

5 элементов сильного промта

  • Роль — кем нейронка должна прикинуться (маркетолог, продажник, коуч)
  • Контекст — что за бизнес, кто ЦА, что уже сделано
  • Задача — чего конкретно надо на выходе
  • Ограничения — стиль, длина, запреты («без буллшита», «без эмодзи»)
  • Формат — структура ответа (список, таблица, 3 варианта)
Чит-код

🔁 Метапромтинг — ИИ пишет промт для ИИ

Тебе лень каждый раз писать полотно? Правильно. Пускай нейронка сама пишет промт, а ты только проверь.

Рабочий мета-промт (копируй)

Ты — промт-инженер мирового уровня.
Моя цель: [опиши задачу 1-2 предложениями].
Я хочу получить на выходе [опиши результат].

Задай мне 5-7 уточняющих вопросов, чтобы собрать весь
нужный контекст. После моих ответов — напиши идеальный
промт по структуре: Роль → Контекст → Задача →
Ограничения → Формат. И дай 2 варианта: базовый и
расширенный с chain-of-thought.

Когда применять

  • Новая задача, не знаешь с чего начать
  • Нужен промт для команды / ассистента
  • Чувствуешь, что «чего-то не хватает» в ответах
  • Хочешь собрать инструкцию для Projects / Cowork

🏋️ Мини-тренировка (сейчас на эфире)

  1. Берём реальную задачу из чата
  2. Скармливаем мета-промт в Клод
  3. Отвечаем на вопросы
  4. Получаем готовый промт и тестим
Техническая база

📦 Что такое контекст и контекстное окно

Контекст — это всё, что нейронка «помнит» в текущем диалоге: твой промт, файлы, предыдущие сообщения, системная инструкция. Контекстное окно — сколько этого влезет.

200K

Claude Sonnet

~150 страниц текста. Базовая рабочая модель для большинства задач.

1M

Claude Opus (1M)

~750 страниц. Для больших проектов, длинных расшифровок, целых книг за раз.

2M

Gemini 2.5 Pro

Рекордсмен по объёму. Для гигантских PDF, видео, целых кодовых баз.

⚠️ Главное, что надо понять

Когда окно заполняется — нейронка начинает забывать начало разговора. Поэтому:

  • Долгие диалоги → начинай новый чат, кидай сводку
  • Важный контекст → выноси в Projects / Cowork как постоянную инструкцию
  • Не пихай «на всякий случай» — пихай то, что нужно под задачу
Основной инструмент курса

⚡ Знакомство с Клодом

Клод за последний месяц перестал быть просто чатом. Сейчас это целая операционка для работы с ИИ. Разбираем что к чему.

Claude.ai (веб-чат)
Обычный чат, Projects (база знаний), Artifacts (код/визуал). Тут живёт основная работа новичка.
Claude Desktop App
То же самое, но ставится на комп. Быстрее запускается, работает с локальными файлами.
Claude Code
Для кода и автоматизации. Через неделю курса ты тоже будешь в нём — делать лендинги, ботов, ассистентов.
Claude Cowork
Новая штука: Клод видит весь твой проект целиком. Не отдельные файлы, а всю папку / бизнес как единое целое.
Routines
Задачи, которые крутятся «пока ноут закрыт». Поставил утром — пришло готовое к вечеру.
Dispatch
Управление Клодом с телефона. Стоишь в пробке — запускаешь ресерч с мобилы.
Customize / Styles
Настраиваешь стиль ответов под себя (ToV, формат, запреты). Один раз — и Клод везде говорит твоим языком.
Короче: за $20/мес ты получаешь то, за что компании платят миллионы. Это база. Это фундамент. Отсюда растёт всё остальное на курсе.
Чем Cowork отличается от всего

🧩 Cowork vs обычный чат vs Claude Code

💬 Обычный чат

Что это: одно окно, один диалог. Закинул — ответил.

Когда юзать: быстрые вопросы, отдельные задачи без связи с другими.

Минус: каждый раз начинаешь с нуля, нет общей памяти.

🧩 Cowork новинка

Что это: рабочее пространство, где Клод видит весь проект целиком — файлы, переписку, контекст.

Когда юзать: когда задача длинная, много материалов, нужна общая картинка.

Плюс: не надо каждый раз объяснять заново. Клод помнит.

⚙️ Claude Code

Что это: терминал + Клод. Для кода, автоматизации, лендингов.

Когда юзать: когда делаешь сайт, бота, ассистента, обрабатываешь кучу файлов.

Это не страшно: на курсе покажем, даже «мамочка в декрете» справляется.

Routers / Dispatch — коротко

Routers — Клод сам выбирает какую модель (Opus/Sonnet/Haiku) кинуть на задачу. Не думаешь — он сам разрулит.
Dispatch — то же самое, но с телефона. Закинул задачу в метро — получил результат дома.

Не все модели одинаково полезны

Переезд с ChatGPT

🧬 Память, контекст и перенос базы из GPT

Как устроена память в Клоде

  • В рамках чата — помнит весь диалог, пока окно не забьётся
  • В Projects — загружаешь файлы + системную инструкцию, они всегда в контексте
  • В Cowork — помнит весь проект, включая историю действий
  • Styles / Customize — помнит твой стиль во всех чатах

Перенос базы из ChatGPT

  1. В GPT: Settings → Personalization → Memory → «Manage memories»
  2. Копируешь всё что там есть
  3. В Клоде создаёшь Project «Обо мне»
  4. В системную инструкцию кидаешь содержимое + свои ДНК Бизнеса / ЦА
  5. Всё. Клод теперь знает про тебя столько же, сколько GPT, и даже больше
💡 Лайфхак: Сделай один Project «Моя база» с файлами: ДНК Бизнеса, ДНК Клиента, ToV, продукты, цены, кейсы. Каждый новый чат в этом проекте уже знает про тебя всё. Экономит часы на объяснениях.
